## Features

- Interactive dashboard with income, expenses, and balance
- Month-wise filtering of transactions
- Spending-by-category visualization (donut chart)
- Smart insights (highest expense, top spending category)
- Add, edit, delete transactions
- Export transactions as CSV
- Dark mode UI with Indigo accent theme
- Deployed on Streamlit Cloud
